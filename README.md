# Setup
Run these commands to start setup
``` bash
curl -fsSL https://raw.githubusercontent.com/ktxdev/setup/main/setup.sh -o setup.sh
```
```bash
chmod +x setup.sh
```
```bash
./setup.sh
```
```bash
rm setup.sh
```
